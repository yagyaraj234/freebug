# Runza logo system

Recommended direction: **Sprint R**.

The open `R` reads as Runza. Its diagonal leg creates forward motion. The inset triangle reads as “run” in developer tooling. Theme light blue connects the mark to the product UI; the underlying mark remains legible in one color.

## Files

- `runza-logo.svg` — primary lowercase wordmark
- `runza-favicon.svg` — lowercase `r` app/favicon mark
- `runza-mark.svg` — compact icon / avatar
- `runza-mark-dark.svg` — compact mark for dark backgrounds
- `variant-01-sprint-r.svg` — recommended monogram
- `variant-02-run-loop.svg` — continuous execution + pass
- `variant-03-test-lanes.svg` — parallel tests converging on proof
- `variant-04-code-pulse.svg` — code-native monitoring
- `variant-05-proof-shield.svg` — safety and confidence
- `variant-06-rz-velocity.svg` — expressive RZ monogram
- `showcase.html` — interactive comparison on light and dark surfaces

## Brand tokens

- Ink: `#0D1117`
- Theme light blue: `#8FA8F6`
- Paper: `#F5F6F1`
- Night: `#11151C`

SVGs use `currentColor` for the main artwork. Override `color` in CSS when embedding. The light-blue run triangle is the only fixed accent in final assets.
